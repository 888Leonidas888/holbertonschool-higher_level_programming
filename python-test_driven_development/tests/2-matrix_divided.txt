El módulo ``2-matrix_divided`` contiene la función:
``matrix_divided``; esta función recvibe un lista de lista
con int o flot, divide cada elemento por el parámetro div y
retorna una nueva lista, con los nuevos valores.
======================================================================

Usando la función ``matrix_divided``
----------------------------------------------------------------------

Este un archivo de pruebas, en modo interactivo.
Empezamos importando ``matrix_divided`` del modulo ``2-matrix_divided``:

Importación:

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Comprobando con tipos int y float:

    >>> m = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(m, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

    >>> m = [[1.5, 2, 8.0], [4, 50.0, 3.6]]
    >>> matrix_divided(m, 2)
    [[0.75, 1.0, 4.0], [2.0, 25.0, 1.8]]

    >>> matrix_divided(m, 2.5)
    [[0.6, 0.8, 3.2], [1.6, 20.0, 1.44]]
